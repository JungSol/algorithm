9장. 동적 계획법 테크닉

* 최적화 문제의 실제 답 계산하기
	* 최적화 문제를 풀 때 최적해의 점수(카운트)를 계산하는 것이 아니라 최적해를 직접 계산해야 할 경우
	(가장 긴 증가부분수열의 길이가 아닌 수열을 찾는경우)
	* 최적화 문제 답 계산하기 레시피
		1. 재귀 호출의 각 단계에서 최적해를 만들었던 선택을 별도의 배열에 저장해 둔다.
		2. 별도의 재귀 함수를 이용해 이 선택을 따라가며 각 선택지를 저장하거나 출력한다.
